
# World Explorer

World Explorer is a Next.js project that allows users to explore countries around the world using data from the REST Countries API.

## Features

- App Router
- File-based Routing
- Shared Layout
- Dynamic Routes
- Server Components
- Client Components
- Real API Data Fetching
- Static Rendering and Caching
- Dynamic Rendering
- Search Functionality
- Country Details Page
- Responsive Design

## API Used

REST Countries API

## Pages

- Home Page
- Countries Page
- Country Details Page
- Search Page
- About Page

## Technologies

- Next.js
- React
- Tailwind CSS

## Run Locally
npm install
cd world-explorer
npm run dev
